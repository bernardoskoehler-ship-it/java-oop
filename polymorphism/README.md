# 🔄 Polymorphism in Java

This folder contains exercises focused on polymorphism in Java.

## 📚 What is polymorphism?
Polymorphism allows methods to behave differently based on the object that is calling them.

## 💡 Concepts practiced
- Method overriding
- Dynamic method behavior
- Use of common interfaces or base classes

## 🎯 Purpose
To understand how different objects can share the same method structure but behave differently.

## 🛠️ Technologies
- Java
