# 🔒 Encapsulation in Java

This folder contains exercises focused on the concept of encapsulation in Java.

## 📚 What is encapsulation?
Encapsulation is the practice of restricting direct access to class attributes and providing controlled access through methods (getters and setters).

## 💡 Concepts practiced
- Private attributes
- Getters and setters
- Data validation
- Controlled access to class data

## 🎯 Purpose
To understand how to protect and manage data within classes.
